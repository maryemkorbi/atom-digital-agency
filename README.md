# Atom Digital Agency - School Project

A complete digital agency website built with **PHP** and **MySQL**.

## 🎨 Color Palette
- 🟦 Deep Navy (#0b0c3d) - Headers, Footer
- 🟧 Peach (#f29f5b) - Accents, Buttons
- 🟩 Dark Green (#1a2e1b) - Secondary backgrounds
- 🟨 Cream (#f0f3ea) - Page background
- ⬜ White (#ffffff) - Cards, Text

## 📄 Pages
- Home (index.php)
- About (about.php)
- Services (services.php)
- Portfolio (portfolio.php)
- Contact (contact.php)

## 🔐 Admin Panel
- URL: /admin/login.php
- Manage portfolio, services, messages, newsletter

## 🛠️ Technologies Used
- PHP 8.2
- MySQL (MariaDB)
- PDO (Prepared Statements)
- Font Awesome Icons
- JavaScript (Vanilla)
- CSS3 (Custom Properties)

## 🚀 How to Run
1. Install XAMPP
2. Copy this folder to htdocs/
3. Import sql/setup.sql in phpMyAdmin
4. Visit http://localhost/atom/
